# Todo App (React + Tailwind CSS)
A clean and minimal Todo application built with React that focuses on proper component design, state management, and a modern UI using Tailwind CSS.

## Features
- Add new todos
- Delete existing todos
- Mark todos as completed
- Filter todos (All / Active / Completed)
- Clean and responsive UI with Tailwind CSS

## Concepts learned and applied
- Component-based architecture in React
- Lifting state up and single source of truth
- Parent-to-child data flow using props
- Updating state immutably
- Controlled inputs
- Rendering lists with keys
- Derived state vs stored state
- Component composition
- Styling React applications using Tailwind CSS
